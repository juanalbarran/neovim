# Neovim

My nix neovim build
It is based in Prima Materia's build.
[PrimaMateria Blog post](https://primamateria.github.io/blog/neovim-editions/#personal-story)

## Build command

### Generic

```bash
nix build .#neovim.<build>
```

### Base

```bash
nix build .#neovim.base
```

### Web

```bash
nix build .#neovim.web
```
